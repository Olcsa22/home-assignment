# Tree Manager Application

This repository contains the implementation of the **Tree Manager Task**.  
The application allows users to create, modify, reorganize, and search nodes within hierarchical tree structures.

The project consists of:

- **Backend:** Java Spring Boot application
- **Frontend:** Angular application

---

# How to start options

## **Option 1:** Visit the webpage 
Open the url `https://www.oliverexperiments.com`

## **Option 2:** Build and run the Application

### Requirements

- **Java 21**
- **Angular 20**
- **Node 22**

### Steps
- Clone the repository:

```bash
git clone https://github.com/Olcsa22/home-assignment.git
cd <repository-folder>
```

- Move into the folder `home_assignment_backend`
- Run `./gradlew build`
- Run `java -jar ./build/libs/home_assignment_backend-0.0.1-SNAPSHOT.jar`
- If it's up and running move into the folder `home_assignment_frontend` in the repository root
- Run `npm i`
- When complete, run `npm run start`
- Open your browser on `localhost:4200`
